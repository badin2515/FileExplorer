# Modern Minimal File Explorer

A beautiful, animated file explorer built with Vite, React, Tailwind CSS (v4), and Framer Motion.

## Features
- **Modern Design**: Glassmorphism, blurred backgrounds, and clean typography.
- **Micro-Animations**: Smooth transitions for folder navigation, hovers, and selections.
- **File Preview**: Click on files to preview details.
- **Responsive**: Adapts to different sizes (though optimized for desktop).

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to the URL shown in the terminal.
